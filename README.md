# **Configuração** ⚙️

Clone o repositório:

```
git clone https://github.com/leomendoncaf/Tarefa8
```

Entre no diretório do projeto:

```
cd nome-do-repositorio
```

Instale as dependências:

```
npm install
```

Inicie o servidor:

```
npm start
```
